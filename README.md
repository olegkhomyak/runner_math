# Runner Math

A simple, fast, and offline-capable Progressive Web App (PWA) for runners to convert paces and calculate race times.

## Features
- **Pace Converter**: Easily switch between min/mi and min/km.
- **Race Calculator**: Instantly see estimated finish times for 5K, 10K, Half Marathon, and Marathon based on your pace.
- **Reverse Pace Calculator**: Enter a target race time to see the required pace.
- **Distance Converter**: Quick conversions between miles and kilometers for common race distances.
- **Offline Ready**: Fully functional without an internet connection. Installable on iOS and Android home screens.
- **Dark Mode**: Supports both light and dark themes based on your system preference.

## Tech Stack
- Vanilla HTML, CSS, and JavaScript
- Zero build tools or dependencies

## Usage
Simply open `index.html` in your browser, or visit the live demo!
